AP1 - Déploiement d'un site web de vente avec Odoo sur Ubuntu Server

Atelier Professionnel 1ère année BTS SIO

Accueil › Ateliers Professionnels › AP1 Odoo

Contexte du projet

Ce projet a été réalisé en 1ère année de BTS SIO (2024-2025) dans le cadre des Ateliers Professionnels. L'objectif était de déployer et configurer le PGI Odoo 16 sur une machine virtuelle Ubuntu Server, puis de tester l'import de données clients générées automatiquement.

Travail en binôme :
Ce TP s'est fait en binôme avec un étudiant option Développement. Mon binôme (développeur) a créé les scripts Python de génération de données CSV, tandis que moi (réseau/système) j'ai géré toute la partie infrastructure : création de la VM, installation d'Odoo, configuration réseau, et tests d'import.
