Installation de Proxmox VE

Configuration du serveur physique et installation complète de Proxmox Virtual Environment

Accueil › Stage 2 › Missions › Mission 2 › Partie 1

Préparation du serveur physique

Le serveur prévu pour Proxmox dispose des caractéristiques suivantes : 40 cœurs, 256 Go de RAM, 24 x 1,2 To de disque dur.

Configuration RAID

Avant de commencer l'installation, je configure les disques du serveur dans le BIOS. J'utilise 5 disques (1,2 To) pour le système, dont un spare (matériel de rechange en cas de panne d'un des disques), en RAID 5 pour l'hébergement de Proxmox ainsi que le stockage des ISO prévues pour concevoir les machines virtuelles.

Configuration du RAID avec 5 disques en RAID 5 pour le système

Configuration RAID : 2 autres blocs de disques de 5 To dont un spare à chaque fois, DATA1 puis DATA2 pour le stockage des machines virtuelles Proxmox.

Installation de Proxmox VE

Installation de l'image Proxmox directement depuis leur site web, puis exécution de cette ISO sur le serveur. Installation de Proxmox VE (Virtual Environment) en mode graphique.

Note importante : L'installation Proxmox que je vais effectuer a été réalisée sur une machine virtuelle afin que je puisse prendre des captures d'écran de meilleure qualité que mes photos. Il est donc normal que certaines données ne correspondent pas aux informations dites auparavant, et c'est aussi pour cela que les adresses IP ne sont pas masquées, je les ai inventées.

Étape 2 : Sélection du disque et configuration

Sélection du disque présent sur la machine (32 Go), sur le serveur Proxmox, j'ai donc choisi le bloc de disques « System » de 5 To

Étape 3 : Localisation et fuseau horaire

Configuration de la localisation et du fuseau horaire

J'ai renseigné le pays ainsi que le fuseau horaire correspondant à ma localisation, et j'ai également passé le clavier en français (azerty) car jusqu'à présent j'étais en qwerty.

Configuration réseau et mot de passe

Mot de passe administrateur

Configuration du mot de passe administrateur

J'ai rentré le même mot de passe que le serveur Proxmox afin d'être sûr de ne pas l'oublier si j'ai l'occasion de réutiliser cette machine virtuelle, puis renseignement d'un email (un faux en l'occurrence, mais lors de l'installation de Proxmox sur le serveur, c'est un vrai qui a été renseigné).

Paramètres réseau

Paramètre Valeur Explication
Carte réseau utilisée nic0 C'est la prise réseau physique du serveur choisie pour piloter la machine
Nom du serveur example.proxmox.kaelig Le nom unique donné à la machine pour l'identifier
Adresse IP du serveur 192.168.100.2 / 24 C'est ce que l'on tape dans notre navigateur pour accéder à Proxmox. /24 correspond au masque de sous-réseau (255.255.255.0)
Passerelle (Gateway) 192.168.100.1 C'est la "porte de sortie" vers les autres réseaux ou Internet (généralement l'adresse du routeur)
Serveur DNS 192.168.200.254 C'est l'annuaire qui permet au serveur de traduire les noms de sites (comme google.fr) en adresses IP
