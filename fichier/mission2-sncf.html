Serveur Proxmox & Supervision Zabbix

Mise en place d'une infrastructure de virtualisation et déploiement d'un système de supervision réseau pour le monitoring temps réel des équipements SNCF Réseau en Bretagne.

Qu'est-ce que Proxmox VE (PVE) ?

Proxmox Virtual Environment (PVE) est une plateforme de virtualisation open-source complète qui combine deux technologies de virtualisation majeures : KVM (Kernel-based Virtual Machine) pour la virtualisation complète et LXC (Linux Containers) pour la conteneurisation légère.

Pourquoi Proxmox pour la SNCF ?
Proxmox permet de centraliser plusieurs machines virtuelles sur un seul serveur physique, optimisant ainsi les ressources matérielles tout en offrant une interface web intuitive pour la gestion complète de l'infrastructure. Cette solution est idéale pour déployer des services critiques comme Zabbix dans un environnement professionnel.

Les fonctionnalités principales de Proxmox incluent :

  • Gestion centralisée via interface web
  • Support de la haute disponibilité (HA)
  • Snapshots et sauvegardes automatisées
  • Migration à chaud des VMs
  • Stockage flexible (local, NFS, iSCSI, Ceph)
  • Gestion réseau avancée (VLAN, bonds, bridges)
